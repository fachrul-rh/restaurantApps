## Cara jalankan aplikasi
1. Klon repositori ini <br />
   ```
   git clone https://github.com/fachrul-rh/restaurantApps.git
   ```
2. Unduh semua dependensi <br />
   ```
   npm install
   ```
3. Jalankan _project_ <br />
   ```
   npm run start-dev
   ```

### Cara menjalankan linter
```
npm run lint
```

### Cara menjalankan Testing
```
npm run test
npm run e2e
```
